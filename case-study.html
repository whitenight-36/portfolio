System idea

One assistant, multiple input modes

Catty was designed to combine text chat, image understanding and voice interaction in one browser interface while keeping conversations organized with SQLite session storage.

Project overview

Catty is a multimodal assistant built around practical AI features. It supports normal text conversations, image-based questions, voice input, object detection, image quality checks and persistent multi-session chat history.

The project shows my ability to combine different AI tools into a working application instead of treating each model as a separate experiment. It also connects my resume strengths: Python, Streamlit, computer vision, NLP, PyTorch, OpenCV and database-backed software development.

Key features

  • Text, image and voice interaction inside a browser-based Streamlit interface.
  • Google Gemini integration for conversational AI responses.
  • Real-time object detection with YOLOv8.
  • Image quality analysis using OpenCV.
  • Visual Question Answering using Salesforce BLIP.
  • Multi-session chat history management using SQLite.
  • Modern responsive user interface for managing multimodal AI interactions.

Technology stack

The project uses Streamlit for the browser interface, Gemini 2.5 Flash for conversational AI, YOLOv8 and PyTorch for object detection, OpenCV for image quality analysis, SpeechRecognition for voice input and SQLite for chat session storage.

1

Input handling

User input can arrive as text, uploaded image or spoken command, giving the assistant multiple ways to understand the request.

2

AI processing

Gemini handles conversation, YOLOv8 handles object detection, OpenCV checks image quality and BLIP supports image question answering.

3

Session memory

SQLite stores session history so conversations can be managed across multiple chats instead of disappearing after a single interaction.

What I learned

This project strengthened my understanding of AI application development, model integration and user-facing software design. It also gave me practical experience with combining object detection, image analysis and conversational AI in one application flow.
